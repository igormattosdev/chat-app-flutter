# minimal_chat_app

This is a flutter project with firebase designed for live chat.

## Functionality
    - Authentication - Login / Register
    - Users
    - Chat between users
    - Theme - Light / Dark mode
  

## Pages
    - Login page
    - Register page
    - User page
    - Chat page
    - Settings page


## Getting Started
    - [Flutter](https://docs.flutter.dev/get-started/install)
    - [Firebase](https://firebase.google.com/)
